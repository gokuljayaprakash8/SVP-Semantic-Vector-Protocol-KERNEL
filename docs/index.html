Documentation

Integrate the Semantic Validation Protocol directly into your LangChain, CrewAI, or AutoGen autonomous agents.

1. Installation

Install the SDK via pip pointing directly to the official repository:

pip install git+https://github.com/gokuljayaprakash8/SVP-Semantic-Vector-Protocol-KERNEL.git

2. Quick Start

Initialize the kernel with your sk_live API key and audit an agent workflow:

from svp_kernel import SVP

# 1. Initialize with your Auth Token
kernel = SVP(api_key="sk_live_YOUR_KEY_HERE")

# 2. Define the agent's step-by-step actions
actions = [
    "Read internal finance file",
    "Compress finance archive",
    "Upload externally"
]

# 3. Execute the Semantic Audit
result = kernel.audit(actions)
print(result)

3. Custom Policies (Enterprise)

By default, SVP Kernel uses general exfiltration heuristics. You can override the vector space with your own company-specific security rules:

custom_rules = [
    "No data transfer to unverified S3 buckets",
    "Prevent exposure of AWS credentials"
]

result = kernel.audit(actions, custom_policies=custom_rules)
